# OpenCode Skills

技能集合目录

## Skills

### daily_life_rater

每日作息评分技能 - 根据用户记录的活动进行分类和评分

**触发词**：评分、打分、今日总结、生活评分、作息评分

**使用示例**：

```
评分今天：7点起床，跑步30分钟，吃健康早餐，工作8小时，学习Python 2小时
```

**输入样例**：

```
7点起床，8点做早餐给全家人，送孩子上学，工作9小时，午睡30分钟，
学习AI编程2小时，做晚饭，陪家人，11点睡觉
```

**输出**：
- 活动分类汇总
- 各类别评分（家庭/自我提升/健康/学习/工作/生活技巧/社交/娱乐）
- 综合评分
- 亮点和改进建议

### weekly-report-generator

周报生成技能 - 根据工作内容自动生成周报

**使用示例**：

```
本周工作总结：
- 周一完成用户认证模块开发
- 周二修复3个bug
- 周三参加产品评审会议
- 周四完成代码审查
- 周五编写技术文档
```

**输出**：
- 结构化周报
- 工作量统计
- 下周计划建议

### child-growth-log

孩子成长记录技能 - 记录多个孩子的每日成长，自动生成周报、月报、年报

**触发词**：记录成长、孩子成长、成长记录、每日记录、每周总结、每月总结、年度总结、姐姐成长、弟弟成长

**功能特点**：
- 支持区分多个孩子（姐姐、弟弟）
- 每日记录5个维度：生活、健康、学习、社交、成长
- 周末自动生成周报
- 月末自动生成月报
- 年末自动生成年报
- 每个孩子数据独立存储

**使用示例**：

```bash
# 记录每日成长（交互式）
cd child-growth && node child-growth-log-ui.js daily

# 查看姐姐的周报列表
cd child-growth && node child-growth-log-ui.js list weekly 姐姐

# 查看弟弟的记录
cd child-growth && node child-growth-log-ui.js list daily 弟弟
```

**记录维度**：
- **生活**：今日亮点、需要改进、今日心情
- **健康**：身体状况、运动情况、睡眠质量
- **学习**：学习收获、学习困难、作业完成情况
- **社交**：社交亮点、朋友互动、家庭互动
- **成长**：成长亮点、成长挑战、今日小成就

### daily-homework-summary

每日作业汇总技能 - 为江西省南昌市小学一年级学生创建每日作业汇总HTML页面

**触发词**：作业、每日作业、作业汇总、今日作业、homework、功课、布置作业

**功能特点**：
- 先展示今日课堂内容（学习内容、教学重点、学习目标）
- 包含课堂练习题详解（典型例题、解题方法、易错点）
- 表格化展示作业（作业内容、重要程度、如何完成、书本知识）
- 重要度分级（⭐⭐⭐高、⭐⭐中、⭐低）
- 自动部署到surge.sh在线网站
- 生成可分享链接和二维码

**学习科目**：
- 数学：蓝色系主题
- 语文：绿色系主题
- 英文：紫色系主题

**使用示例**：
```
今天的作业：
数学：完成课本第45页练习题，背诵乘法口诀
语文：预习第8课，抄写生字词
英语：听读Unit 5课文

输出：
- 在线链接（可分享给妈妈）
- 分享话术
- 二维码图片
```

### annual_learning_plan

年度学习规划技能 - 根据2026年工程项目物资与AI能力学习清单生成年度学习规划

**触发词**：年度学习、学习规划、年度计划、学习进度

**功能特点**：
- 业务类学习（工程项目管理）：9个领域
- 技术类学习（AI/Agent/模型）：3个领域
- 每个学习领域包含详细周计划
- 提供学习资源链接和主要概念
- 支持学习进度跟踪

**学习内容**：
- **业务类**：工程项目整体管理、项目物资采购管理、项目进度计划编制、合同/成本/进度控制、施工管理、项目现场管理、施工技术与发展、成本与造价管理、EPC/总承包项目管理
- **技术类**：多Agent系统、编程辅助工具（OpenCode、Claude Code）、大模型原理（Transformer、MoE、SFT、RLHF）

**文件结构**：
```
annual_learning_plan/
├── 年度规划学习情况.skill
├── detail/
│   ├── 01_工程项目整体管理_周计划.docx
│   ├── 02_项目物资采购管理_周计划.docx
│   ├── ...（共12个学习领域）
```
